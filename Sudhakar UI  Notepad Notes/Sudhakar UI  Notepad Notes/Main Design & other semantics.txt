                            dialog
- It is used to create a popup.
- Dialogs are usually hidden.
- You can open and display when ever required.
- Controlling dialog dynamically requires JavaScript.

Syntax:
    <dialog>
        … your content …
    </dialog>

- To show dialog add "open" attribute

Syntax:
    <dialog open>
        … your content …
    </dialog>

                            Code

- It represents are code snippet. [code block]
- It is to make the code in page SEO friendly and Compiler friendly.

                            Var
- It represents a variable.
- Variables are storage locations in computer programming.

                            Pre
- It represents pre-formatted content in page.
- It keeps the content alignment and spacing exactly as defined in code.

Syntax:
    <code> </code>
    <var> </var>
    <pre> </pre>

                        Line Break & Blank Space

 <br>             for line break.
 &nbsp;             for blank space

                        Sample Text

- <samp> is used for sample text.
- It is used for "note and comments" for content in your page.

Syntax: Line Break
      <div>
        Welcome <br>
        To <br>
        HTML
    </div>

Syntax: Blank Space

    H&nbsp;&nbsp;T&nbsp;&nbsp;M&nbsp;&nbsp;L

Syntax: Pre-formatted

    <pre>
        .. your code block …
    </pre>
