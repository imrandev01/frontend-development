                            Data List

- It is a collection of terms & definitions.
- List is defined with "<dl>"
- Terms are defined with <dt>
- Definitions are defined with <dd>

Syntax:
    <dl>
       <dt> Term </dt>
       <dd> Definition </dd>
    </dl>

Ex:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">

    <title>Document</title>
    <style>
        dt {
            background-color: black;
            color:white;

            padding: 5px;

        }

        <dt>Styles</dt>
        <dd>It defines styles.</dd>
        <dt>JavaScript</dt>
        <dd>It manipulates DOM.</dd>
    </dl>
   
</body>
</html>

                        Details & Summary
- Details is a container that can show or hide any content dynamically.
- Summary is a caption for details.

Syntax:
    <details>
        <summary> Title </summary>

        …. your content ….

    </details>

- Details is by default hidden in state. You can open details on page load by using the attribute "open".

Syntax:
    <details open>
    </details>

Ex:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        summary {
            background-color: black;
            color:white;
            padding: 5px;
        }
        details div {
            background-color: black;
            color:white;
            padding: 20px;
            margin-top: 10px;
        }
        details {
            margin-bottom: 20px;
        }
    </style>
</head>
<body>
    <h1 align="center">Frequently Asked Questions</h1>
    <details open>
        <summary>What is Netflix?</summary>
        <div>
            <p>Netflix is a streaming service that offers a wide variety of award-winning TV shows, movies, anime, documentaries and more – on thousands of internet-connected devices.</p>
            <p>You can watch as much as you want, whenever you want, without a single ad – all for one low monthly price. There's always something new to discover, and new TV shows and movies are added every week!</p>
        </div>
    </details>
    <details>
        <summary>How much does Netflix cost?</summary>
        <div>
            <p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Provident tenetur numquam ipsam eius minus, reprehenderit nulla, facere repudiandae possimus exercitationem quasi, molestiae esse modi quod quisquam accusamus. Illo, eveniet dolor?</p>
        </div>
    </details>
</body>
</html>

                        Fieldset & Legend

- Fieldset is a container with frame.
- Legend is a caption for container.
- It is used to keep all relative content together.

Syntax:
    <fieldset>
        <legend> Title </legend>
       
        …. your content ….

    </fieldset>

                        Figure & Caption

- Figure is a container used to encapsulate any graphic with caption.
- It keeps the content SEO friendly.
- Caption can be used as keyword for searching a picture.

Syntax:
    <figure>
       
        … your image …

        <figcaption> keyword </figcaption>

    </figure>

Ex:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        .women-fashion {
            background-image: url("./images/women-fashion.jpg");
            background-size: cover;
            width: 200px;
            height: 300px;
        }
        .men-fashion {
            background-image: url("./images/men-fashion.jpg");
            background-size: cover;
            width: 200px;
            height: 300px;
        }
        figure {
            border:1px solid black;
            padding: 10px;
            width: 200px;
        }
        figcaption {
            background-color: maroon;
            color:white;
            padding: 5px;
            text-align: center;
            margin-top: 10px;
        }
        body {
            display: flex;
        }
    </style>
</head>
<body>
    <figure>
        <div class="women-fashion">

        </div>
        <figcaption>Women Fashion</figcaption>
    </figure>
    <figure>
        <div class="men-fashion">

        </div>
        <figcaption>Men Fashion</figcaption>
    </figure>
</body>
</html>

                        Images in HTML

- Images are of various types in computing system.
- Web will not support all types of images.
- The most recommended image types for web are:

Image                                MIME Type                Extension
--------------------------------------------------------------------------------------------------------------------
APNG
[Animated Portable Network Graphics]        image/apng                .apng

PNG
[Portable Network Graphics]                image/png                .png

JPEG
[Joint Photographic Expert Group]            image/jpeg                .jpg, .jpeg, .jfif

GIF
[Graphic Interchange Format]            image/gif                    .gif

TIFF
[Tagged Image File Format]                image/tiff                    .tiff

Bitmap                                image/bmp                .bmp

Microsoft Icon                            image/icon                .ico

SVG                                    image/svg-xml                .svg
[Scalar Vector Graphics]                application/xml

Web Picture                            image/webp                .webp

FAQ: What is MIME type?
Ans:  MIME is Multi purpose Internet Mail Extension
         A MIME type refers to the file type that comprises of various extensions for
     different situations. Browser can understand the file only by MIME type.
