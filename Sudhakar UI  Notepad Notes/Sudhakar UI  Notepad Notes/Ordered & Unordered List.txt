                        Ordered List

- It is used to configure auto numbering for a list of items.
- Numbering updates automatically when you add or delete items.
- List is defined with <ol>
- List Item is defined with <li>

Syntax:
    <ol>
       <li> Item-1 </li>
       <li> Item-2 </li>
    </ol>

Attributes:
1. type        : It defines numbering type as "1, a, A, I, i"

              <ol type="A">

2. start        : It specifies the initial value to start with. It refers to level number.

                <ol type="A"  start="5">
                <ol type="1"   start="5">

3. reversed    :  It is used to reverse the numbering style in descending order.
   
                 <ol type="1"  reversed>

- You can configure nested list.
- A child list must be with in "<li>" element.

FAQ's:
1. How many levels of nested numbering can be defined?
A. There is no specified limit. You have 5 numbering types, which you have to repeat if there are more levels.

2. How to display list items side by side?
A. By using "display:flex" for ordered list.

Ex:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        .option-list {
            display: flex;
            margin-top: 10px;
        }
        .option-list li {
            margin-right: 40px;
        }
    </style>
</head>
<body>
    <ol type="1">
        <li>__________ elements will not have end token.
            <ol type="a" class="option-list">
                <li>Normal Elements</li>
                <li>Void Elements</li>
                <li>RC Data Elements</li>
                <li>Raw Text Elements</li>
            </ol>
        </li>
    </ol>
</body>
</html>

3. How to display list in multiple columns?
A. By using  "display:grid"  or  "columns" in CSS

Syntax:
    ol {                                        ol {
         display:grid;                    (or)            columns: 2
         grid-template-columns: 6fr 6fr;                }
     }

4. How to create a scrollable list?
A. By using CSS overflow attribute

Syntax:
    ol {
        height: 50px;
        border:1px solid gray;
        overflow:auto;
    }

Ex:

<html lang="en">

5. How to remove numbering for list?
A. By using the attribute "list-style" set to "none"

Syntax:
    ol {
        list-style: none;
    }

                        Unordered List
- It configures a bulleted list.
- Bulleted list contains a symbol instead of number.
- List is defined with <ul>
- Items are defined with <li>

Syntax:
    <ul>
     <li> Item-1 </li>
     <li> Item-2 </li>
    </ul>

- <ul> type can be  "disc, circle or square"

    <ul type="circle">
    <ul type="square">

FAQ:
1. How to define a custom symbol?
A. By using CSS attribute  "list-style-image"
    By using 3rd party icons classes.

Ex:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        ul {
            list-style: none;
            list-style-image: url("./icons/start.gif");
            font-size: 30px;
        }
    </style>
</head>
<body>
    <ul>
        <li>HTML</li>
        <li>CSS</li>
        <li>JavaScript</li>
    </ul>
</body>
</html>
