Transforms & Transitions

                   
                          Keyframes & Animations
- Keyframe is a container with element that changes according to state and situation.
- Animation is created with a set of keyframes.
- Keyframe animation comprises of
    a) Initial State
    b) Final State
    c) Steps

Syntax:
        @keyframes   TransitionName
        {
            from {  }            // Initial
            50% {  }            // step
            to { }                // final
        }
   
- Keyframes are assigned to any element by using animation properties

        animation-name
        animation-duration
        animation-property
        animation-delay
        animation-direction
        animation-iteration-count
        animation-timing-function

Syntax:
    element {
        animation-name: KeyFrameName;
        animation-duration: 2s;
        animation-iteration-count: 2 | infinite;
        animation-direction: reverse | alternate;
        animation-timing-function: ease-in;
     }
