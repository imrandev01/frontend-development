9. Form Classes

    .form-control            textbox, password, url, email, number, date, textarea
    .form-range            range
    .form-select            dropdown
    .form-control-color        color
    .form-control-file        file
    .form-label            label
    .form-check-input        radio, checkbox
    .form-check-label        labels for checkbox & radio
    .form-switch            for radio & checkbox container

Syntax:
    <input type="text"  class="form-control">
    <select class="form-select">
    <input type="range" class="form-range">
    <input type="radio" class="form-check-input">
