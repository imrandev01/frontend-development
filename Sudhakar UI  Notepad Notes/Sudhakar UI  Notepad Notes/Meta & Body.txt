                            Meta
- Meta refers to "Meta Data".
- Meta data is information about your page. [data-about-data].
- It is used to make the page SEO friends and Responsive.

Meta Responsive:

    <meta  name="viewport"  content="width=device-width, initial-scale=1">

- To test responsive page download and install  "Mobile Simulator" extention for Chrome browser.

    viewport                : It refers to screen / device type and dimensions.

    initial-scale            : It refers to zoom% for content on device.
                          1=100%

    <meta  http-equiv="refresh"   content="4">


    refresh                : It reloads the page content every few seconds.
                          1 = 1second


Meta SEO Friendly:

    <meta  name="keywords"  content="some keywords to find your page">
    <meta  name="description" content="something about your page to show in SEO">
    <meta  charset="utf-8">

     UTF        = Unicode Transformation Format
     8            = 8bit memory size required to read the chars.
                = 8bit is used for english
                = 16bit memory
                = 32bit memory






Syntax:
<head>
  <meta charset="utf-8">
  <meta name="keywords" content="some words">
  <meta name="description" content="some description">
  <meta http-equiv="refresh" content="5">
  <meta name="viewport" content="width=device-width, initial-scale=1">
</head>



                            Body Section

- The <body> section defines content to load directly into browser.
- The contents of body require a round-trip everytime when requested.

Attributes:
1. bgcolor        : It sets background color for page
2. text            : It sets color for text.

HTML Colors:
a) Color Name
b) Shade Name
c) Hexa Decimal Code

        <body bgcolor="red">            color name
        <body bgcolor="darkred">        shade name

Hexa Code
    a) 3 chars code with "#"
    b) 6 chars code with "#"

            #RGB                
            #RRGGBB
