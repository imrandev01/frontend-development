Offcanvas
- It is similar to modal but have various directions.

Classes:
    .offcanvas
    .offcanvas-start
    .offcanvas-end
    .offcanvas-top
    .offcanvas-bottom
    .offcanvas-header
    .offcanvas-body

Syntax:
    <div class="offcanvas offcanvas-start" id="menu">
        <div class="offcanvas-header">
        </div>
        <div class="offcanvas-body">
        </div>
    </div>
Attributes:
    data-bs-target
    data-bs-toggle

Ex:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Offcanvas</title>
    <link rel="stylesheet" href="../node_modules/bootstrap-icons/font/bootstrap-icons.css">
    <link rel="stylesheet" href="../node_modules/bootstrap/dist/css/bootstrap.css">
    <script src="../node_modules/bootstrap/dist/js/bootstrap.bundle.js"></script>
</head>
<body class="container-fluid">
    <div class="btn-toolbar bg-dark p-2">
        <button data-bs-target="#menu" data-bs-toggle="offcanvas" class="bi bi-justify btn btn-light"></button>
    </div>
    <div class="offcanvas offcanvas-start" id="menu">
        <div class="offcanvas-header bg-dark text-white">
            <h2>Amazon</h2>
            <button class="btn btn-close-white btn-close" data-bs-dismiss="offcanvas"></button>
        </div>
        <div class="offcanvas-body">
            <ul>
                <li>Home</li>
                <li>About</li>
                <li>Contact</li>
            </ul>
        </div>
    </div>
</body>
</html>

                           Nav with Tabs
- It is a dynamic navigation between tab based content in page.

Classes:
     .nav
     .nav-tabs
     .nav-pills
     .nav-menu
     .nav-item
     .nav-text
     .nav-link
     .tab-content
     .tab-pane
     .active     [ it is defined for nav-item ]

Syntax:
    <ul class="nav nav-tabs">
       <li> <a href="#tab-id" data-bs-toggle="tab"> Item1 </a>  </li>
    </ul>
    <div class="tab-content">
       <div class="tab-pane" id="">
       </div>
    </div>

                               Navbar
- It is used for responsive navigation.
- It can change according to device width and orientation.

Classes:
    .navbar
    .navbar-brand
    .navbar-toggler
    .navbar-toggler-icon
    .navbar-expand-lg | sm
    .navbar-nav
    .nav-menu
    .nav-item
    .nav-link
    .nav-text
    .navbar-collapse
    .navbar-dark | light
    .bg-{contextual}

Note: To make responsive always use "container-fluid" for your content.

Syntax:
    <nav class="navbar navbar-expand-lg navbar-dark bg-dark">
       <div class="container-fluid">
        <button class="navbar-toggler"> </button>
       </div>
       <div class="navbar-collapse">
         <ul class="navbar-nav">
         </ul>
       </div>
    </nav>
Ex:
<!DOCTYPE html>
