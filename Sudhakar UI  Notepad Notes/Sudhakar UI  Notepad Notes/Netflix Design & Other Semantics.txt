CSS Colors:
- In CSS you can apply colors using following techniques

1. Color Name
    {
      color : red;
    }

2. Shade Name
    {
     color: gold;
    }

3. Hexa Decimal Code
    {
      color: #f00 (or)  #ff0000;
    }

4. RGB Method
    {
      color: rgb(redValue, greenValue, blueValue);     0 to 255 [RGB]
    }

5. RBGA Method (A=Alpha=Transparent=0-1)
    {
      color:rgba(255,0,0,0.5);    RGBA
    }

 6. HSL Method (Hue Saturation Lighting)
    {
      color:hsl(hue, sat, light);
    }

    hue        : It is an angle from 0 to 360deg
              0-120      Red
              120-240 Green
              240-360 Blue

    sat        : 0 to 100% gray shade

    light        : 0 to 100% brightness

Syntax:
    {
     color: hsl(140deg, 50%, 50%);
    }

7. HSLA (Alpha)
    {
     color: hsla(140deg, 50%, 50%, 0.5);
    }

                            Footer
- It is a semantic used to display content at bottom margin of page.
- Typically footer contains
    - Contact Address
    - Services
    - Copyrights
    - Quick links
    - Social Bar
      etc..

                        Address

- It is used to keep contact details.
- Address semantic will make your website contact details SEO friendly.

                        Aside
- It represents the content that is not present in current page or website.
- It redirects the user to another website.
- It moves aside the current website.
- Usually it is used for social links, ads etc.

Summary
- Section
- Main
- Address
- Aside
- Footer
