Margin Attributes:

    margin        :  short hand for all directions.
                   [top, right, bottom, left]

                   {
                    margin: 0px 0px 0px 0px;        4 directions individual
                   }

                   }

    margin-top
    margin-bottom

Border Image Attribute:

        border-image                : It sets image as border with size, offset, style

Syntax:
    {
      border : 10px solid transparent;
      border-image: url("path")  stretch|space  1-100;         1-100 is offset
    }

Note: You can't set border-image without the border attribute.

Ex:
   p {
       border:24px solid transparent;
       border-image: url("../public/images/border.jpg") space 70;
       margin: 50px;
       padding: 20px 5px 20px 10px;
     }

                          CSS Positions
- The CSS "position" attribute provides following values
   
    a) Static
    b) Absolute
    c) Fixed
    d) Relative
    e) Sticky

Static
- It keeps element according to the normal flow in document.
- It will not allow the element to change its position using top, left, right and bottom values.
- It is the default position for element.

Absolute:
- It removes element from the normal flow in document.
- It allows to keep at specified position using top, right, left and bottom.
- It keeps the element with regard to the content in page.
- It moves the element along with content.

Fixed:
- It removes element from the normal flow in document.
- It allows to change position.
- It keeps the element width regard to browser window and not to content in page.
- It will not move along with the content.

Relative:
- It defined for parent element to keep all child elements relative to parent.

Sticky:
- It keeps element according to the normal flow in document.
- It can scroll along with content in page.
- However it can stick to browser at specified position.
