                            Hyperlink in HTML
- Hyper refers to "beyond".
- A hyper link navigates the user to any specified location.
- Hyperlink can be clickable text, picture or graphic.
- Navigation is classified into 2 types
    a) Intra Document
    b) Inter Document
- Intra document navigation enables redirection from one location to another within the same page.
- Inter document navigation enables redirection from one page to another.
- Hyperlink is created by using anchor element <a>.

Intra Document Links:
1. You have to define target location with "id", which is a unique reference for elements in page.

            <header id="top">
            <h2 id="html">
            <img id="pic">

2. You have to set ID in URL by using <a>  "href" attribute.

            <a href="#top">  Top </a>
            <a href="#html"> HTML </a>
            <a href="#pic"> Pic </a>

FAQ: How to remove underline for hyperlink?
Ans: By using CSS "text-decoration" set to "none".

Syntax:
    a{
       text-decoration: none;
    }

FAQ: How to highlight any link target if you can't scroll to the specified location?
Ans:  By using CSS "target" class.

Syntax:
    container {
        // effects before target
    }
    container:target {
        // effects on target
    }

    - Make sure that container is defined with "Id"    
    - <a> href is referring to ID.

Inter Document Links:
- It defines a hyperlink that redirects the user to any

        a) file
        b) url
        c) application

Link to File:
- You can link to any file : html, doc, xls, pdf, txt, jpg etc.
- But every file can't open in browser.
- Browser can open only the files that have supported plugin's in browser.
- If plugin is not installed in browser, then file will download.

Syntax:    
    <a href="./docs/file.pdf"> PDF </a>
    <a href="./images/pic.jpg"> Image </a>
    <a href="./docs/word.docx"> Word Document </a>

FAQ: How to configure a link to file that always downloads the target file?
Ans:  HTML 5 provides "download" attribute for <a> element, which downloads the file always.

Syntax:
    <a href="./docs/file.pdf"   download="newname">  Download File </a>

Link to URL:
- You can define a valid remote URL for navigation.
- Make sure that the URL is defined with protocol and domain.

            www.amazon.in        // not good
            https://www.amazon.in    // good

Syntax:
    <a href="https://www.amazon.in"&gt; Amazon </a>


Link to Application:
- Anchor element support some applications, which you can invoke in browser.
- It includes  email application, skype, phone application etc.

        tel            : phone app
        mailto        : email app
        skype        : skype app

Syntax:
    <a href="emai@gmail.om"> Email </a>
    <a href="tel:phonenumber"> Phone </a>
