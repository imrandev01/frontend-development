Styles in HTML

- Styles provide a set of attributes to make HTML more interactive and responsive.
- Style uses an object to refactor the code.
- Object contains relative data and logic.
- Style objects are used to make HTML interactive.
- Styles can defined for HTML elements in 3 ways

    a) Inline Styles
    b) Embedded Styles
    c) External Style Sheet

Inline Styles:
- The style object is defined for every element individually using "style" attribute.
- It is faster in rendering.
- It is not good for reusability.

Syntax:
    <h2  style="attribute:value; attribute:value">  </h2>

Ex:
    <h2  style="color:red; font-family:arial">  HTML </h2>

Embedded Styles:
- In embedded technique the style object is defined in a <style> container.
- So that it is good for reusability and separation.
- Embedded is good for extensibility.

Syntax:
        <style>
             selector
              {
            attribute:value;
            attribute:value;
              }
        </style>

- It is slower than inline.
- You can't access embedded styles from another page.
- Embedded is only for the context of current page.

FAQ's:
1. Where to embed the <style> container, in <head> or in <body> ?
A. You can embed in both sections.

2. What is difference between styles in head and styles in body?
A. Styles in head are used to save round-trip.
     Styles in body are for one time rendering.

3. If styles are defined for element in both head and body then which set of attribute will apply?
A. If different attributes are used then all will apply.
    If same attribute is used then the latest will apply.  [LIFO]

4. What is the media type for embedded styles?
A. Media type is used to define style for various media devices, which include
    a) screen
    b) print    
    c) speech

Syntax:
    <style media="all | screen | print | speech">

    </style>

Ex:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style media="screen">
        .ticket {
            border:1px solid black;
            padding: 5px;
            background-color: yellow;
        }
        dt{
            font-weight: bold;
        }
        dl {
            display: grid;
            grid-template-columns: 3fr 9fr;
        }
    </style>
    <style media="print">
        .ticket {
            background-color: none;
            border:1px solid black;
            padding: 5px;
        }
        button, aside {
            display: none;
        }
    </style>
</head>
<body>
   <div class="ticket">
      <aside>
        <img src="../public/images/a1.jpg" width="100" height="100">
      </aside>
      <h2>Ticket</h2>
      <dl>
        <dt>Departure</dt>
        <dd>Hyderabad</dd>
        <dt>Destination</dt>
        <dd>Delhi</dd>
        <dt>Time</dt>
        <dd>10:20PM</dd>
      </dl>
      <button onclick="window.print()">Print</button>
   </div>
</body>
</html>

5. What is the MIME type of styles?
A. MIME is to define content type, so that browser can understand the object type.
    The styles objects MIME type is "text/css".

Syntax:
    <style type="text/css"  media="all">

    </style>

6. Is it mandatory to define MIME type of stylesheet linked with <link> element?
A. No. It is not recommended.

    <link rel="stylesheet"  href="some_path"  type="text/css">       Not Good


External Style Sheets:
- In this technique styles are defined in a separate style sheet that have extension ".css".
- So that you can reuse across pages.
- Using an external style sheet will increase the number of requests to page and also load time.
- CSS refers to "Cascading Style Sheet"

Ex:
1. src/styles/ticket.css

.ticket {
    border:1px solid black;
    padding: 5px;
    background-color: yellow;
}
dt{
    font-weight: bold;
}
dl {
    display: grid;
    grid-template-columns: 3fr 9fr;
}

2. home.html

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <link rel="stylesheet" href="../src/styles/ticket.css">
</head>
<body>
   <div class="ticket">
      <aside>
        <img src="../public/images/a1.jpg" width="100" height="100">
      </aside>
      <h2>Ticket</h2>
      <dl>
        <dt>Departure</dt>
        <dd>Hyderabad</dd>
        <dt>Destination</dt>
        <dd>Delhi</dd>
        <dt>Time</dt>
        <dd>10:20PM</dd>
      </dl>
      <button onclick="window.print()">Print</button>
   </div>
</body>
</html>
