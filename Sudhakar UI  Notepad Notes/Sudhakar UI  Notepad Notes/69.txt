Object Manipulations:

1- How to access all keys from object?
A. 
    a) for...in
    b) Object.keys()

Syntex:
    for (var key in object)
    {

    }

    Object.keys(ObjectName).map()


<script>
    var product = {
        id: 1,
        title: "Sumsung TV",
        price: 45000.44,
        stock: true
    }
    for (var key in product) {
        document.write(key + "<br>");
    }
    Object.keys(product).map(function(key){
        document.write(`<li>${key}</li>`);
    })
</script>

3- How to delete a key?
A 
    By using JS "delete" operator

Syntex
        delete object.key;
        delete object.price;

4- How to find a key?

A   By using "in" operator

Syntex: 
        "key" in objectName;            //true if object have the given key

EX- 
    <script>
    var product = {
        id: 1,
        title: "Sumsung TV",
        price: 45000.44,
        stock: true,
        rating: 4.5
       }
    if ("rating" in product){
        document.write(`"Rating" :${product.rating}`);
    } else {
        document.write(`Rating" property does not exist in product object.`);
    }
</script>

6- How to know the data type of a value stored in key ?
a   By using "typeof" operator

Syntax
        typeof object.key                   //returns data type
        typeof object[key]
        typeof product[price]

<script>

                                    How to display list item

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Menu using map()</title>

    <script>
        var menu = [
            { Category: "Electronics", product: ["TV", "Mobile", "Watch"] },
            { Category: "Fashion", product: ["Kids", "Men", "Women"] }
        ];

        function bodyload() {

            var ol = document.querySelector("ol");

            // map() returns category <li> elements
            var categoryLis = menu.map(function (item) {

                var categoryLi = document.createElement("li");
                categoryLi.innerText = item.Category;

                // map() returns product <li> elements
                var productLis = item.product.map(function (prod) {
                    var li = document.createElement("li");
                    li.innerText = prod;
                    return li;   // IMPORTANT: return
                });

                var ul = document.createElement("ul");

                // append returned product lis
                productLis.map(function (li) {
                    ul.appendChild(li);
                    return null; // map requires return, value ignored
                });

                categoryLi.appendChild(ul);
                return categoryLi; // IMPORTANT: return
            });

            // append returned category lis
            categoryLis.map(function (li) {
                ol.appendChild(li);
                return null;
            });
        }
    </script>
</head>

<body onload="bodyload()">
    <ol></ol>
</body>
</html>
