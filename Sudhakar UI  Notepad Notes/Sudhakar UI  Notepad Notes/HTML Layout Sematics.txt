FAQ: What is a <div>?
Ans:
    - It refers to a division.
    - It is a container used to keep all related content together.
    - Technically it is used to refactor content.
    - Refactor is the process of extracting relative content into a division.

FAQ: What is a <span>?
Ans:
    - span refers to part of existing content.
    - It is a container that keeps content along with existing but allows to access
      individually.

Adding Icons into Project:
- There are various free and premium icon providers for web development
    a) bootstrap icons
    b) font awesome icons
    c) telerik icons
    d) mui icons
    etc..

Setup Bootstrap Icons:
1. Open Terminal in VS Code [ Project Terminal ]

2. Run the command

    > npm  install  bootstrap-icons --save

3. It adds a new folder into project by name "node_modules"

4. Node_Modules is the folder that contains all library files that you install using NPM.

5. Bootstrap icons are kept in a file  "bootstrap-icons.css"

Path:
    node_modules/bootstrap-icons/font/bootstrap-icons.css

6. Icons are stored under the reference of class names  in ".css" file

        .bi bi-house
        .bi bi-bell
        .bi bi-facebook
        etc..

        icons.getbootstrap.com

7. To use icons in any HTML page you have to link the CSS file

<head>
<link rel="stylesheet"  href="../node_modules/bootstrap-icons/font/bootstrap-icons.css">
</head>
<body>
    <span class="bi bi-house"> </span>
    <div  class="bi bi-twitter-x"> </div>
</body>

Body Layout Semantics:

1. <header>
- It defines the content to display at top margin of page.
- Typically header of website comprises of
        * Brand Logo
        * Brand Title
        * Navigation
        * Shortcut Links
        * Search Bar
          etc..
2. <nav>
- It is for defining navigation area in a page.
- It comprises of items used to navigate between content in site.
