Image Advanced Attributes:

1. crossorigin         :  It refers to CORS [Cross Origin Resource Sharing].
                   It is used for images, which are accessed from remote locations.
                   It allows to set restrictions for image.
                       
                        a) allow-anonymous
                        b) use-credentials

Syntax:
 <img src="some-remote-url"  crossorigin="anonymous | use-credentials">

Ex:
<img width="500" height="400" crossorigin="use-credentials" src="https://apod.nasa.gov/apod/image/2405/BlackHole_Simonnet_960.jpg"&gt;

2. decoding        :  It controls image loading along with other content in page.
                   It can define
                    a) sync        : blocking  [blocks other content]
                    b) async        : unblocking [along with other content]
                    c) auto        : changes according to situation

Syntax:
    <img src="..."  decoding="async | sync | auto">

3. Importance        : It sets priority for image.
                  Priority includes
                    a) low
                    b) high
                    c) auto
Syntax:
    <img src="..."  importance="low | high | auto">


4. srcset            : It is used to configure multiple images.
                  An image element can embed multiple images.
                  However images will rotate[change] with CSS attributes.


Syntax:
    <img srcset="set of images">

FAQ: How to keep image SEO friendly?
Ans:  Enclose image in <figure> with <figcaption>

Syntax:
    <figure>
        <img src="pic.jpg" width="200" height="200">
        <figcaption> Title </figcaption>
    </figure>

CSS Float for Images:
- "float" is a CSS attribute used to float content left or right.
- It controls the content wrapping along with alignment.
- "float" may effect other elements in container, hence you have to use "clear" as CSS attribute to discard float.

    {
      float: left | right;
      clear:left | right | both;
    }

Ex:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        .pic2 {
            float: left;
        }
        .pic1 {
            float: right;
        }
        p {
            clear: left;
        }
    </style>
</head>

FAQ: What is difference between align and float?
Ans : Float is a responsive attribute of CSS. It can be controlled dynamically.
     Align is basic HTML attribute used for text-wrapping. It is not responsive.
