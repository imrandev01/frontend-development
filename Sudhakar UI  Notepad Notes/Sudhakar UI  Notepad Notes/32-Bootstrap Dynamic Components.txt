Bootstrap Dynamic Components

Note: Dynamic components require "bootstrap.bundle.js"  script file.
      Link the script file into HTML page

      <script src="../node_modules/bootstrap/dist/js/bootstrap.bundle.js"> </script>

1. Alerts
- It is an embedded message box in page.
- It is along with existing content in page.
- It is used to notify messages in page.

Classes:
    .alert
    .alert-dismissible
    .alert-{contextual}
    .alert-link
    .alert-title
    .alert-subtitle

Syntax:
    <div class="alert alert-dismissible"> </div>

Dynamic Attributes:
    data-bs-dismiss         : It is defined for button to close the component on click

Syntax:
    <button data-bs-dismiss="alert"> OK </button>

Ex:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Alerts</title>
    <link rel="stylesheet" href="../node_modules/bootstrap-icons/font/bootstrap-icons.css">
    <link rel="stylesheet" href="../node_modules/bootstrap/dist/css/bootstrap.css">
    <script src="../node_modules/bootstrap/dist/js/bootstrap.bundle.js"></script>
</head>
<body class="container-fluid">
    <h2>Alerts</h2>
    <div class="alert alert-success alert-dismissible">
        <button class="btn btn-close" data-bs-dismiss="alert"></button>
        <h3 class="alert-title">Registered Successfully</h3>
        <p class="alert-subtitle">Welcome to our website - Verify the email</p>
        <a class="alert-link" href="#">Help?</a>
        <br><br>
        <button data-bs-dismiss="alert" class="btn btn-success">OK</button>
    </div>
    <div class="alert w-25 alert-info alert-dismissible">
        <button class="btn btn-close" data-bs-dismiss="alert"></button>
        <img src="../public/images/a1.jpg" width="200" height="200">
        <h3>45% OFF</h3>
    </div>
</body>
</html>

2. Modal
- It is dialog that pops up.
- Content is in the backdrop and dialog is over the content.

Classes:
    .modal
    .modal-dialog
    .modal-content
    .modal-header
    .modal-body
    .modal-dialog-centered
    .modal-dialog-scrollable
    .modal-fullscreen
    .modal-footer

Syntax:
    <div class="modal">
       <div class="modal-dialog">
           <div class="modal-content">
            … header, body ,footer …
        </div>
      </div>
       </div>

-  Every modal must have an "ID" reference.
-  Every modal is hidden in page.
-  You can open specific model with button or link click.
-   It requires attributes.

Dynamic Attributes
    data-bs-toggle        : which component to open, it refers to component name
    data-bs-target        : It refers to component "ID"
    data-bs-dismiss    : which component to close

Syntax:
    <div  class="modal"  id="login"> </div>

    <button data-bs-toggle="modal" data-bs-target="#login">
    <a href="#login"  data-bs-toggle="modal">

- To open model with transition use ".fade" class.

    <div class="modal fade"> </div>

3. Collapse
- It is used to show or hide any content in page dynamically.

Classes:
    .collapse
    .show

Syntax:
    <div class="collapse show" id=""> </div>

Attributes:
    data-bs-target
    data-bs-toggle

Ex:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Collapse</title>
    <link rel="stylesheet" href="../node_modules/bootstrap-icons/font/bootstrap-icons.css">
    <link rel="stylesheet" href="../node_modules/bootstrap/dist/css/bootstrap.css">
    <script src="../node_modules/bootstrap/dist/js/bootstrap.bundle.js"></script>
</head>
<body class="container-fluid">
    <button class="btn btn-dark w-100 mt-3" data-bs-toggle="collapse" data-bs-target="#q1" >What is Netflix?</button>
    <div class="collapse show bg-dark p-3 mt-2 text-white" id="q1">
        <p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Ipsam libero adipisci non nostrum, corporis laborum accusamus porro eligendi aspernatur earum molestiae ducimus maxime perspiciatis eos quam voluptates sed quae culpa.</p>

</body>
</html>

4. Accordion
- It is a combination of multiple items using "Mutex" mechanism.
- You can select one item to open and show, where all other items will close.

Classes:
    .accordion
    .accordion-item
    .accordion-header
    .accordion-body
    .accordion-collapse
    .accordion-button

Syntax:
    <div class="accordion"  id="">
          <div class="accordion-item">
           … header, body with collapse …
          </div>
          <div class="accordion-item">
          </div>
    </div>

Attribute:
    data-bs-target
    data-bs-toggle
    data-bs-parent    : It is defined for all accordion-collapse items.
                      It refers to ID of accordion.

    <div class="accordion-collapse collapse"  id="q1" data-bs-parent="#faqs">

Ex:
<!DOCTYPE html>
