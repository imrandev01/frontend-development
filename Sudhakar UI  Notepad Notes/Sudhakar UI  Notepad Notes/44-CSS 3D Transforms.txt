                          CSS 3D Transforms
- 3 Dimensional Transforms comprises of X, Y & Z axis.
- 3D requires a perspective to view the content.

a) Translate3D
    translateX()
    translateY()
    translateZ()

b) Sclae3D
    scaleX()
    scaleY()
    scaleZ()

c) Rotate3D

d) Skew3D


Note: Transforms are not supported across all browsers, hence you need various CSS
      plugins to configure.

        -webkit            -webkit-transform
        -moz            -moz-transform
        -o                -o-transform
        -ms                -ms-transform

Syntax:
       {
          transform:scaleX(200px);
          -webkit-transform:scaleX(200px);
          -moz-transform:scaleX(200px);
          -o-transform:scaleX(200px);
          -ms-transform:scaleX(200px);
      }

                           CSS Transitions

transition                    : short hand for all effects
transition-duration            : time take to complete the transform
transition-delay            : the delay time for starting transform
transition-property            : the properties that use the timings and effects
transition-timing-function    : pre-defined transition effects

Ex:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        div {
            width: 50px;
            height: 50px;
            background-color: red;
            transition-duration: 5s;
            transition-delay: 1s;
            transition-property: width;
            transition-timing-function: steps(4);
        }
        div:hover {
            width: 1000px;
            background-color: yellow;
            transition-duration: 2s;
            transition-delay: 1s;
            transition-property: width;
            transition-timing-function: steps(20);
        }
    </style>
</head>
<body>
    <div></div>
</body>
</html>
