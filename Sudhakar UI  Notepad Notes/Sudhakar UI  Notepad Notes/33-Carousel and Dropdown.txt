                              Carousel
- It configures sliding and fading content in page.

Basic Carousel Classes:

        .carousel
        .carousel-inner
        .carousel-item

Syntax:
    <div class="carousel">
        <div class="carousel-inner">
                <div class="carousel-item">
            </div>
             … multiple items …
        </div>
    </div>

- Make sure that any one carousel item is active. You have to use ".active" class.

     <div class="carousel-item  active"> </div>

- Carousel don't have any default animation, you have to define animation by using the attribute  "data-bs-ride". It is used to start animation.

    <div class="carousel"  data-bs-ride="carousel">

- Carousel can have 2 types of animation styles
       
        .carousel-fade
        .slide

   <div class="carousel slide">
   <div class="carousel carousel-fade">

Ex:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <link rel="stylesheet" href="../node_modules/bootstrap-icons/font/bootstrap-icons.css">
    <link rel="stylesheet" href="../node_modules/bootstrap/dist/css/bootstrap.css">
    <script src="../node_modules/bootstrap/dist/js/bootstrap.bundle.js"></script>
</head>
<body class="container-fluid">
    <div class="carousel slide" data-bs-ride="carousel">

Carousel Controls:
- Controls allows to navigate between items.
- It includes previous and next item.

Classes:
        .carousel-control-prev
        .carousel-control-prev-icon
        .carousel-control-next
        .carousel-control-next-icon

Attributes:
        data-bs-slide="prev | next"
        data-bs-target="carousel_id"

- You can apply dark controls by using
        .carousel-dark                [ old version]
        data-bs-theme="dark"        [ new version]

Carousel Indicators:
- You can implement random navigation using indicators.
- They allow to navigate to any specific item directly.

Class:
    .carousel-indicators

Attributes:
    data-bs-slide-to="indexNumber"    [0 is first item]
    data-bs-target="carousel_id"

- Make sure that any one indicator is active, always keep the item and indicator synchronized.

                            Dropdown
- Bootstrap dropdown is good for a static list of items.
- If list is not regularly changing and contains complex contents then better use a bootstrap dropdown.

Classes:
    .dropdown
    .dropend
    .dropstart
    .dropup
    .dropdown-toggle
    .dropdown-menu
    .dropdown-item
    .dropdown-item-text
    .dropdown-divider
    .dropdown-menu-dark [obsolete]  data-bs-theme="dark"   [new]

Ex:
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Dropdown</title>
    <link rel="stylesheet" href="../node_modules/bootstrap-icons/font/bootstrap-icons.css">
    <link rel="stylesheet" href="../node_modules/bootstrap/dist/css/bootstrap.css">

</head>
<body class="container-fluid">
