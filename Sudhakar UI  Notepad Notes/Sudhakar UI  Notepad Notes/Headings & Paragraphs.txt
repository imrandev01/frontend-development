CSS Overflow:
- It is used to control the overflowing content in a container.
- If container have fixed width and height then content inside container will overflow.
- You can set overflow as
    a) hidden
    b) scroll
    c) auto

Syntax:
    div {
           height: 200px;
           overflow: scroll;
     }

                        Headings in HTML

- Headings are defined with <hn> element.
- "n" refers to level number of heading from 1 to 6.

Syntax:
    <h1> Heading-1 </h1>
    <h2> Heading-2 </h2>
    ……….
    <h6> Heading-6 </h6>

FAQ's:
1. Why you need a heading element if you can design similar using styles or other elements?
A. To make headings in page SEO friendly. Search engine can identify and make a list of topics in page only when defined with headings.

2. What are the default styles applied to heading?
A. font-size, font-weight & display:block

3. How to remove any specific default style?
A. You can remove any specific style for HTML elements using CSS "unset" value.

Syntax:
    h1 {                            h1        
          font-weight:unset;            {
          display:unset;                  all : unset;
    }                            }

4. Can we change styles defined for heading?
A. Yes. You change existing styles or apply new styles.

Note:
 - Don't use too many headings in a page. It may SPAM the page.
 - Don't use headings for highlighting any word or sentence in a paragraph.

5. How to inherit parent element styles?
A. By using CSS "inherit" value.

    {
      border:inherit;
      padding:inherit;
    }

6. How to set default state for inherited styles?
A. By using CSS "initial" value

    {
      color:initial;
    }

7. What is difference between "unset" & "initial"?
A. If it is inherited style then you have to use "initial".
     If it is default style then you have to  use "unset".

                      Paragraphs & Blockquotes
- Paragraph <p> element
- Blockquote <blockquote> Element.
- Paragraph is to keep content relative.
- Blockquote is to keep the summary about content.

Syntax:
    <p> some text </p>
    <blockquote> some text </blockquote>

    blockquote {
        font-style:italics;
        padding-left: 100px;
            padding-right:100px;
        text-align:justify;
    }

FAQ's:
1. How to set first line indent for paragraph or blockquote?
A. By using CSS "text-indent" attribute

Syntax:
    blockquote {
        text-indent: 50px;
    }

2. How to set Drop Cap?
A.
    - Access the first letter of paragraph by using CSS class "::first-letter"
    - Apply float, font size & style.

Syntax:
    p::first-letter {
        float:left;
        font-size:25px;
        font-family:Arial;
    }

3. How to set space between lines , words & chars?
A. By using CSS attributes
        line-height        : for line spacing
        word-spacing        : space between words
        letter-spacing        : space between chars
