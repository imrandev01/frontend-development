FAQ: How to display paragraph with ellipsis?
Ans:     By using the CSS attributes

        white-space            : It can display text without wrapping.
        overflow                : It can hide overflowing text
        text-overflow            : It sets ellipsis for text overflow.
        width                : It sets fixed width for paragraph

Syntax:
    p {
        width: 250px;
        white-space: nowrap;
        overflow:hidden;
        text-overflow: ellipsis;
    }
