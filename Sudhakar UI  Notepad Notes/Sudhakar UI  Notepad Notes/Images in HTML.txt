1. PNG & APNG
- High Definition
- More Pixel Depth
- Good Quality
- Uses more memory
- Loads slow
- It is good for downloads.

2. JPEG
- High Definition
- Less Pixel Depth compared to PNG
- Good Quality
- Compressed Format
- Less memory.
- Loads Fast
- Good for on screen presentation.

3. GIF
- Less Pixel Depth than JPG
- Only 256 colors
- Not HD
- Poor Image Quality
- Memory same as JPG
- Load Fast
- Can be animated
- Good for icons, logos, bullets, lines, patterns etc.

4. TIFF
- Sliced image format
- It is used for scanned documents.
- HD
- Need More memory than PNG
- More Pixel than PNG

5. ICON
- Microsoft Icons
- Used for Favicon [.ico]
- Max "32 x 32" recommended


6. SVG
- It is not pixel based.
- It is vector based.
- Need more memory
- High Quality
- HD
- High Resolution at various zoom levels.
- It is XML format image. [Extensible Markup Language]
- Mostly used for Icons and Architectural diagrams.

7. Bitmap
- Image in binary format.
- It is not much recommend in modern web.
- It is good to store in database.
- It is good in COM marshal.
  [Common Object Model]
  Binary to Object => Object to Binary
- Need more memory
- Loads slow


8. Webp
- Web Picture
- It a picture embedded into Webpage.
- You can't extract or save image individually.


                        Image Element

- HTML <img> element is used to embed image into page.

Attributes:
1. src        : It defines the image path and name.

2. alt        : It defines the alternative text to display when image fails to load.

3. title        : It defines text to display when mouse is over.

4. width        : It sets width for image.

5. height        : It sets height for image.

6. border        : It sets border with specified pixels.

7. align        : It aligns left or right with text wrapping.

8. vspace        : Vertical padding space

9. hspace    : Horizontal padding space

Syntax:
 <img src="./images/fashion.jpg"  width="200" height="300" border="3" align="left" alt="Women Fashion"  title="Offer Ends Today"  hspace="20" vspace="30">

FAQ: What is fluid Image? How to define a fluid image?
Ans:  Fluid image can adjust according to the parent window or container.
     You can define fluid images with size in "%" instead of pixels.

Syntax:
    <img src="./images/pic.jpg"  width="50%"  height="30%">
