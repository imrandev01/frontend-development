The While Loop:-
- TheWhile Loop:
- It is used when it is not sure about the number of iterations and iteration counter may
change dynamically.
- It executes the statements only when the given condition evaluates to true.
Syntax:
while(condition){
    statements;
    counter;
    }

The Do While loop:
- It similar to while but ensures that statements will execute at least once even when the condition is false.

Syntax:
    do{
        statements;
        counter;
    } while(condition);


 <script>
    var i = 11;
    do {
        document.write(i+ "<br>");
        i++;
    } while (i<=10);
 </script>
