Array with Spread Operator:
-You can use "..." which is a spread Operator to copy the elements of an Array into another.
-It allows to sepead the values of a collection into individual references.

Syntex:
        var one =["A","B"];
        var two =[one];             //two have 1 element 
        var two =[one ,"C" ,"D"]    //two have 2 Elements
        var two =[...one ,"C" ,"D"] //two have 4 elements 

Syntex
        var one =["A" ,"B"];
        var two =["C","D"];
        var three =[...one , two];      //3 elements
        var three =[...one, ...two];    //4 elements
        var three =[one, two];          //2 elements

                            Object Type.
-Object is used to keep all related data and logic together.
-"Alan Kay" introduced the concept of object into computer programming in early 1960's.
-Technically object is a key-value pair.
-key is always string type and value can be any type.

Syntax:
        var obj ={
            "key":value,
            "key":value
        }

-Value can be 
        a) primitive
        b) non-primitive
        c) function
Syntex
    var obj = {
        "key":number,
        "key":string,
        "key":new Array,
        "key":{object},
        "key":function(){}
                }

- If object comprises of only data without any action (function) defined, then it is referred as "JSON". 
- JSON is JavaScript Object Notation, It is a format for data. (data is maintained in a format colled JSON).
        *JSON is light weight 
        *JSON is faster in transporting and rendering
        *JSON doesn't require COM to Marshal [Converting object to binary & vice versa]
        *JSON can flow through firewalls.
        *JSON is not infected by VIRUS.

Syntex:
    {
        "Name":"TV",
        "Price":34000.44,
        "Stock":true,
        "ShippedTo":["Delhi","Hyd"],
        "Rating",{"Rate":4.6, "Count":4500}
    }

-Object values are accessed with reference of "Key".

Syntex:
    var product = {"Name":"TV", "Price":56000};

    product.Name;           //return TV
    product.Price;          //return 56000

-You can assign new values if declared with "var or let".

    product.Name =newname;
    product.Price = newprice;
