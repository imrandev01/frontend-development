Dropdown List
- It allows to select any one option from a group of choices.
- List is defined by using
    select
- List contains collection of items defined using
    option
- Options can be grouped by using
    optgroup

Syntax
    select
       optgroup label=title
        option Item-1 option
        option Item-2 option
      optgroup
    select

- Every option have 3 attributes

    a) value             It defines the value to submit
    b) selected         It keeps the option selected on load
    c) disabled         It will not allow to select the option.

    option value=  selecteddisabled

- Every option is RC data type, It will not allow icons & images as value or text.
    [only alphabet, number & special chars allowed]

                            ListBox
- Dropdown transforms into list box with attributes
    a) multiple
    b) size

Syntax
    select            = dropdown

- List box allows to select multiple options.

Syntax
    select size=2 multiple

                                Meter
- It is used to show grade meter.
- It is controlled with attributes
    a) min
    b) max
    c) value
    d) low
    e) high

Syntax
    meter min= max= value= low= high= meter

- Colors for meter

        low, high     are set to zero             green
        more gap between low-high         gold
        less gap between low-hight         red

                            Progress
- It is used to show the status of any task performed by page.
- It includes time elapsed and time remaining.

Syntax
    progress progress

Attributes
    a) min
    b) max
    c) value

- It is controlled dynamically by using JavaScript.

Syntax
    progress min=1 max=100 value=40 progress


                            Buttons
- Buttons are used to confirm user actions.
- Buttons are 2 types
    a) Generic Buttons
    b) Non Generic Buttons

- Generic Buttons
    a) submit         It can submit form data.
    b) reset         It will reset form data to default.

    HTML 4
        input type=submit  value= 
        input type=reset  value= 

    HTML 5
        button type=submit Text  Images  Icons button
        button type=reset Text  Image  Icon button

- Non Generic button

     HTML 4
        input type=button  value= 
   

    HTML 5
        button type=button Text  image  icons button

Note If button type is not defined then the default type is submit for HTML 5 buttons.
